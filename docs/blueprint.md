# **App Name**: Tweet Weaver

## Core Features:

- Fetch News Headlines: Fetch the latest news headlines from a news API based on a given topic and maximum number of results.
- Summarize Articles to Tweet Length: Generate a tweet-length summary (<=280 characters) for each article, including a short summary, topic hashtag, and original source link.
- Tweet Rewriting Tool: Utilize a generative AI tool to rewrite the generated tweet to sound more human and engaging. This tool reasons whether to suggest modifications or not, without direct human intervention.
- Save Tweets to File: Save all generated tweets to a timestamped .txt file for later use and archival.
- Print Tweets to Console: Print the generated tweets to the console for immediate viewing.
- Topic and Tone Control: Allows users to set the topic, maximum results and the tone (Neutral, Breaking and Casual) of the articles and tweet that would be generated.
- UI Theme Toggle: Allows the user to switch between light and dark UI themes for personalized viewing experience.

## Style Guidelines:

- Primary color: Midnight Blue (#191970) for a professional and trustworthy feel.
- Background color: Off-white (#F5F5DC) in light mode and Dark gray (#333333) in dark mode.
- Accent color: Sky Blue (#87CEEB) to highlight key elements.
- Font pairing: 'Inter' for body text (sans-serif) and 'Literata' for headlines (serif), for a balanced, readable interface.
- Use simple, modern icons to represent different categories and actions.
- Clean, organized layout with clear separation between control panels, articles preview, and tweet drafts.
- Subtle animations for loading and tweet generation.
